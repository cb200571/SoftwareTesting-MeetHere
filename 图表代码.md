# 图表生成代码

使用以下代码在 Mermaid 在线编辑器（https://mermaid.live）中粘贴即可生成图片。

---

## 图1：用例图

```mermaid
graph TB
    Admin((管理员))

    subgraph 数据看板
        UC1[查看数据看板]
        UC11[查看概览数据<br/>4个数字卡片]
        UC12[查看订单状态分布<br/>环形饼图]
        UC13[查看月度订单趋势<br/>折线图]
        UC14[查看热门场馆排行<br/>柱状图]
    end

    Admin --> UC1
    UC1 --> UC11
    UC1 --> UC12
    UC1 --> UC13
    UC1 --> UC14
```

## 图2：时序图

```mermaid
sequenceDiagram
    actor 管理员
    participant 浏览器
    participant Controller as AdminStatisticsController
    participant DAO as OrderDao/UserDao/VenueDao
    participant DB as MySQL数据库

    管理员->>浏览器: 1. 点击"数据统计"菜单
    浏览器->>Controller: 2. GET /statistics_manage
    Controller-->>浏览器: 3. 返回HTML空壳页面

    浏览器->>Controller: 4. GET /statistics/overview.do
    Controller->>DAO: 5. countAllOrders()
    DAO->>DB: SELECT count(*) FROM order
    DB-->>DAO: 返回订单总数
    DAO-->>Controller: 返回订单总数
    Controller->>DAO: 6. sumTotalRevenue()
    DAO->>DB: SELECT IFNULL(sum(total),0) WHERE state=3
    DB-->>DAO: 返回营收
    DAO-->>Controller: 返回营收
    Controller->>DAO: 7. countAllUsers()
    DAO->>DB: SELECT count(*) FROM user
    DB-->>DAO: 返回用户数
    DAO-->>Controller: 返回用户数
    Controller->>DAO: 8. count()
    DB-->>DAO: 返回场馆数
    DAO-->>Controller: 返回场馆数
    Controller->>DAO: 9. countByState(1~4)
    DAO->>DB: SELECT count(*) WHERE state=1,2,3,4
    DB-->>DAO: 返回各状态订单数
    DAO-->>Controller: 返回各状态订单数
    Controller-->>浏览器: 10. JSON: 8个数字
    浏览器->>浏览器: 11. 渲染4个数字卡片
    浏览器->>浏览器: 12. 渲染饼图

    浏览器->>Controller: 13. GET /statistics/monthlyOrders.do
    Controller->>DAO: 14. countOrdersByMonth()
    DAO->>DB: SELECT DATE_FORMAT, count(*) GROUP BY month
    DB-->>DAO: 返回月度数据
    DAO-->>Controller: 返回月度数据
    Controller-->>浏览器: 15. JSON: 月份+数量数组
    浏览器->>浏览器: 16. 渲染折线图

    浏览器->>Controller: 17. GET /statistics/venueRank.do
    Controller->>DAO: 18. findTopVenues()
    DAO->>DB: SELECT venueID, count(*) GROUP BY venueID LIMIT 5
    DB-->>DAO: 返回TOP5场馆ID+数量
    DAO-->>Controller: 返回TOP5场馆ID+数量
    Controller->>DAO: 19. getOne(venueID) ×5
    DAO->>DB: SELECT * FROM venue WHERE venueID=?
    DB-->>DAO: 返回场馆名称
    DAO-->>Controller: 返回场馆名称
    Controller-->>浏览器: 20. JSON: 场馆名+数量数组
    浏览器->>浏览器: 21. 渲染柱状图

    浏览器-->>管理员: 22. 页面完整展示
```

## 图3：活动图

```mermaid
flowchart TD
    A([开始]) --> B[管理员点击左侧菜单"数据统计"]
    B --> C[浏览器发送GET请求到/statistics_manage]
    C --> D[后端返回HTML空壳页面]
    D --> E[JS自动发出4个AJAX请求]
    E --> F[后端查询数据库返回JSON数据]
    F --> G[前端渲染4个数字卡片]
    G --> H[前端渲染饼图]
    H --> I[前端渲染折线图]
    I --> J[前端渲染柱状图]
    J --> K([结束])
```

## 图5：类图 — 新增和修改的类

```mermaid
classDiagram
    class AdminStatisticsController {
        -OrderDao orderDao
        -UserDao userDao
        -VenueDao venueDao

        +getOverview() Map

        +getVenueRank() List~Map~
    }

    class OrderDao {
        <<interface>>
        +findByOrderID(int) Order
        +findAllByState(int, Pageable) Page~Order~
        +findAudit(int, int) List~Order~
        +findAllByUserID(String, Pageable) Page~Order~
        +updateState(int, int) void
        +countAllOrders() int ★新增
        +countByState(int) int ★新增
        +sumTotalRevenue() int ★新增
        +countOrdersByMonth() List~Object[]~ ★新增
        +findTopVenues() List~Object[]~ ★新增
    }

    class UserDao {
        <<interface>>
        +findByUserIDAndPassword(String, String) User
        +findByUserID(String) User
        +findAllByIsadmin(int, Pageable) Page~User~

        +countAllUsers() int ★新增

    class VenueDao {
        <<interface>>

    }

    AdminStatisticsController --> OrderDao : 调用
    AdminStatisticsController --> UserDao : 调用
    AdminStatisticsController --> VenueDao : 调用
```

## 图7：页面结构图

```mermaid
graph TB
    subgraph statistics.html
        subgraph 顶部导航栏
            NAV[layout/header.html]
        end
        subgraph 左侧菜单
            LEFT[layout/left.html<br/>★新增"数据统计"菜单项]
        end
        subgraph 主内容区
            subgraph 概览卡片
                C1[总订单数]
                C2[总营收]
                C3[总用户数]
                C4[总场馆数]
            end
            subgraph 图表区域
                PIE[订单状态饼图<br/>id=pieChart]
                LINE[月度趋势折线图<br/>id=lineChart]
                BAR[热门场馆柱状图<br/>id=barChart]
            end
            FOOTER[layout/footer.html]
        end
    end

    AJAX1[AJAX: /statistics/overview.do] --> C1
    AJAX1 --> C2
    AJAX1 --> C3
    AJAX1 --> C4
    AJAX1 --> PIE
    AJAX2[AJAX: /statistics/monthlyOrders.do] --> LINE
    AJAX3[AJAX: /statistics/venueRank.do] --> BAR
```
